# Covid Visualizer React

Stack used:
 - ReactJS v18
 - Typescript
 - TailwindCSS
 - Chart.js
 - React-chartjs-2
 - React-select
 - React-loader-spinner
 - Axios

 Data source: https://covid.ourworldindata.org/data/owid-covid-data.json
 Data git source: https://github.com/owid/covid-19-data/tree/master/public/data
